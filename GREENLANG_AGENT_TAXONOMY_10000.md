## Agent Classification System

```
Agent ID Format: GL-[Category]-[Subcategory]-[Number]
Example: GL-CALC-CARBON-001 (Carbon Calculator Agent #001)

Categories:
- CALC: Calculators & Computation
- MON: Monitoring & Sensors
- OPT: Optimizers & Controllers
- PRED: Predictive & Forecasting
- COMP: Compliance & Regulatory
- REP: Reporting & Documentation
- INT: Integration & Connectors
- ANAL: Analytics & Insights
- AUTO: Automation & Workflow
- COMM: Communication & Engagement
```
