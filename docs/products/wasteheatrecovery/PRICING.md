## 3. Performance-Based Options

### Gain-Share Model

**Structure:**
- Base subscription at 50% of standard rate
- 20% of verified energy savings (annual)
- Minimum 3-year commitment
- Third-party verification of savings

**Example - Professional Edition:**
- Standard pricing: $75,000/year
- Gain-share base: $37,500/year
- If savings = $500K: Performance fee = $100,000
- Total: $137,500 (vs. $75,000 standard)
- If savings = $200K: Performance fee = $40,000
- Total: $77,500 (vs. $75,000 standard)

**Eligibility Requirements:**
- Minimum $1M annual energy spend
- Existing heat recovery infrastructure
- Commitment to implement recommendations
- Access for verification measurements

### Project Success Fee

**For capital project identification:**
- 5% of verified first-year savings from implemented projects
- Paid upon project completion and verification
- Separate from ongoing subscription
- Minimum $10,000 per project
