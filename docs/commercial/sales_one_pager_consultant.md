## Typical consultant workflow

```
1. Create sub-tenant for "Acme Inc." engagement
2. Upload Acme's utility bills via /export bulk intake
3. Apply Acme-specific methodology overrides (e.g. their own
   Scope 2 market-based REC certificates)
4. Run /match + /resolve across their full scope
5. Export audit bundle — signed, timestamped, ready for assurance
6. Deliver to Acme as a sealed package + PDF summary
```
