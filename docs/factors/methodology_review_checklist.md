# Methodology review checklist (P5)

Use before promoting a factor or batch from **preview** to **certified**.

1. **Scope and category** — GHG Protocol scope and category consistent with numeric vectors.
2. **Boundary** — combustion vs WTT vs market-based vs residual mix correctly tagged.
3. **Geography** — Code matches source table region; subnational vs national intentional.
4. **GWP** — AR5 vs AR6 and horizon documented; matches product default policy.
5. **Units** — Numerator/denominator align with GreenLang unit ontology after conversion.
6. **Temporal validity** — valid_from / valid_to within source publication intent.
7. **License** — `license_class` and redistribution match rights matrix sign-off.
8. **Lineage** — Raw artifact ID and parser run ID resolvable.
9. **Duplicates** — Merge policy applied; no conflicting twins in certified set.
10. **Regulatory tags** — CBAM / SB253-relevant flags set where applicable.
