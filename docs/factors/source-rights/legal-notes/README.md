# Source-Rights Legal Notes

Each file in this directory is the engineering-side legal-note record
for one upstream factor source. The note captures:

* the source's authoritative publication URL,
* the licence terms we rely on,
* the redistribution scope we have verified,
* attribution requirements,
* any known commercial restrictions,
* the CTO Phase 1 delegated approval (until the named
  Compliance/Security Lead countersigns).

These notes are **not** a substitute for signed legal opinions where
the licence terms are commercial / restricted (Ecoinvent, IEA,
NIES, etc.). Those are added as `legal_signoff.evidence_uri` only
once Compliance/Security obtains the signed counsel memo.

The single source of truth for legal status remains
`greenlang/factors/data/source_registry.yaml#legal_signoff`. A note
file in this directory MUST exist for every source whose
`legal_signoff.evidence_uri` points here.
