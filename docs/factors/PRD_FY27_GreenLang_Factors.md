# PRD: GreenLang Factors FY27

## Summary

GreenLang Factors is the **auditable emissions-factor operating layer** for developers, consultants, and climate platforms—not a flat spreadsheet. FY27 delivers a versioned catalog, deterministic APIs, provenance and lineage, QA gates, source-aware licensing, update/watch workflows, and hybrid matching (rules + retrieval) with evaluation harnesses.

## Buyers and jobs-to-be-done

- **Developers:** embed calculations with reproducible, pinned releases.
- **Consultants:** search, cite, and defend factor choices under CBAM, SB 253 (Scope 1–2 first cycle), and evolving GHG Protocol guidance.
- **Platforms:** integrate factor infrastructure without maintaining dozens of source calendars.

## FY27 success metrics (three numbers)

| Metric | Target |
|--------|--------|
| Catalog rows (normalized) | 100,000+ with explicit status labels |
| QA-certified, default API | 40,000–60,000 |
| Connector-backed / licensed | 20,000–40,000 (never implied as openly redistributable) |

## In scope (v1)

- Source registry, rights matrix, ingestion, normalization, lineage, QA, releases, read APIs, match API (hybrid), update watch, pilot kit, enterprise hooks.

## Out of scope (v1)

- Full PCF/DPP workflow, deep proprietary LCA redistribution without license, carbon credit MRV as primary object.
