# GreenLang v1 Milestone Calendar

## Week-by-Week Milestones

- Week 1: Phase 0 charter sign-off and owner assignment
- Weeks 2-3: Phase 1 contracts and fixtures stabilized
- Weeks 3-5: Phase 2 CLI/runtime semantics and conformance tests
- Weeks 5-6: Phase 3 security/policy baseline enforcement
- Weeks 6-8: Phase 4 determinism/audit/observability standards applied
- Weeks 8-10: Phase 5 multi-app CI and regression lanes enforced
- Weeks 10-11: Phase 6 docs and UAT execution
- Weeks 11-12: Phase 7 hardening, RC, soak, go/no-go

## Milestone Owners

- Runtime: Core Platform
- Contracts: Schema and Pack Governance
- Security/Policy: Governance and Security
- Determinism/Audit: Provenance team
- CI/CD: DevEx
- Docs/UAT: Documentation and Enablement
