# CC1 — Control Environment

**Owner:** People Ops + CISO. **Review cadence:** annual.
**Legal review required:** code-of-conduct language.

## Controls

### CC1.1 — Commitment to integrity and ethical values

- Evidence: signed Code of Conduct acknowledgements (People Ops HRIS).
- Collection: manual.
- Artifacts to attach each audit:
  - [ ] Current Code of Conduct PDF (versioned).
  - [ ] HRIS export of signed acknowledgements (all active employees).
  - [ ] Ethics hotline call log summary (if any reports filed).

### CC1.2 — Board independence and oversight

- Evidence: board roster + audit committee charter.
- Collection: manual.

### CC1.3 — Organizational structure

- Evidence: current org chart with role-responsibility mapping.
- Collection: manual.

### CC1.4 — Workforce competence

- Evidence: security training completion logs (annual).
- Collection: semi-automated (HRIS export + SIEM cross-check).
- Artifacts:
  - [ ] Training LMS report for current period.
  - [ ] New-hire checklist template (onboarding security sign-off).

### CC1.5 — Accountability

- Evidence: performance review policy; corrective-action SOP.
- Collection: manual.
