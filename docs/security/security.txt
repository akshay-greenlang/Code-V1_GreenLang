# GreenLang Climate OS - Security Response
#
# We take security seriously. If you believe you have found a security
# vulnerability in GreenLang, please report it to us through our
# Vulnerability Disclosure Program (VDP).
#
# What to expect:
# - Acknowledgment within 24 hours
# - Initial assessment within 5 business days
# - Regular updates on remediation progress
# - Bounty awards for valid findings
#
# Scope:
# - GreenLang Cloud Platform (app.greenlang.io)
# - GreenLang API (api.greenlang.io)
# - GreenLang Open Source Projects (github.com/greenlang)
#
# Out of Scope:
# - Social engineering attacks
# - Physical security issues
# - Denial of service attacks
# - Spam or social media hijacking
#
# Thank you for helping keep GreenLang and our users secure.
