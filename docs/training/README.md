# GreenLang Process Heat Agents - Training Materials

## Overview

Welcome to the GreenLang Process Heat Agents training program. This comprehensive curriculum is designed to help operators, administrators, and developers effectively deploy, manage, and optimize GreenLang's AI-powered process heat management systems.

## Training Paths

### Path 1: Operator Training (Recommended: 2-3 days)

For process engineers, plant operators, and control room personnel who will use GreenLang on a daily basis.

| Module | Duration | Prerequisites |
|--------|----------|---------------|
| [Fundamentals](./fundamentals.md) | 4 hours | None |
| [Operator Training](./operator_training.md) | 8 hours | Fundamentals |
| Video Tutorials 01-03 | 2 hours | None |

**Learning Outcomes:**
- Understand GreenLang architecture and core concepts
- Navigate the operator interface effectively
- Respond to alarms and alerts appropriately
- Troubleshoot common operational issues
- Optimize process heat performance

### Path 2: Administrator Training (Recommended: 3-4 days)

For IT administrators, DevOps engineers, and system managers responsible for deploying and maintaining GreenLang.

| Module | Duration | Prerequisites |
|--------|----------|---------------|
| [Fundamentals](./fundamentals.md) | 4 hours | None |
| [Administrator Training](./administrator_training.md) | 12 hours | Fundamentals |
| [Quick Start - Production Deployment](../quickstart/production_deployment.md) | 4 hours | Administrator Training |
| Video Tutorials 01-05 | 3 hours | None |

**Learning Outcomes:**
- Install and configure GreenLang in production environments
- Manage users, roles, and security policies
- Implement backup and disaster recovery procedures
- Tune system performance for optimal operation
- Integrate with existing enterprise systems

### Path 3: Developer Training (Recommended: 5 days)

For software developers and integration specialists building custom solutions with GreenLang.

| Module | Duration | Prerequisites |
|--------|----------|---------------|
| [Fundamentals](./fundamentals.md) | 4 hours | None |
| [Administrator Training](./administrator_training.md) | 12 hours | Fundamentals |
| [API Documentation](../api/README.md) | 8 hours | Administrator Training |
| [Integration Guides](../integrations/README.md) | 8 hours | API Documentation |
| Video Tutorials 01-05 | 3 hours | None |

**Learning Outcomes:**
- Develop custom agents and modules
- Build integrations with external systems
- Extend ML capabilities with custom models
- Create automated workflows and pipelines
- Implement custom safety compliance modules

## Training Environment

### Sandbox Access

All trainees receive access to a sandbox environment for hands-on practice:

- **URL:** https://sandbox.greenlang.io
- **Duration:** 30 days from enrollment
- **Resources:** Pre-configured agents, sample data, test scenarios

### System Requirements

For local training environments:

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | 4 cores | 8+ cores |
| RAM | 8 GB | 16+ GB |
| Storage | 50 GB SSD | 100+ GB SSD |
| OS | Windows 10/11, Ubuntu 20.04+, macOS 12+ | Latest LTS versions |
| Docker | 20.10+ | Latest stable |
| Python | 3.9+ | 3.11+ |
