# GreenLang Video Tutorial Series

## Overview

This directory contains scripts for the GreenLang video tutorial series. These scripts are designed for professional video production and cover essential topics for getting started with GreenLang Process Heat Agents.

## Video Series

| Video | Title | Duration | Audience |
|-------|-------|----------|----------|
| [01](./01_introduction.md) | Introduction to GreenLang | 15 min | All users |
| [02](./02_installation.md) | Installation and Setup | 20 min | Administrators |
| [03](./03_first_agent.md) | Creating Your First Agent | 25 min | Operators, Engineers |
| [04](./04_ml_features.md) | Machine Learning Features | 30 min | Engineers, Data Scientists |
| [05](./05_safety_compliance.md) | Safety and Compliance | 20 min | All users, Compliance Officers |

## Target Audience

- **Operators:** Focus on Videos 01, 03, 05
- **Engineers:** All videos
- **Administrators:** Focus on Videos 01, 02, 05
- **Executives:** Video 01 only

## Production Guidelines

### Visual Style

- **Resolution:** 1920x1080 (1080p)
- **Frame Rate:** 30 fps
- **Aspect Ratio:** 16:9
- **Color Palette:** GreenLang brand colors (green, white, dark gray)

### Audio Requirements

- **Narration:** Professional voice-over
- **Background Music:** Subtle, non-distracting
- **Sound Effects:** Minimal, for emphasis only

### Screen Recordings

- **Browser:** Chrome or Firefox (latest version)
- **Screen Size:** 1920x1080
- **Cursor:** Visible with highlighting for clicks
- **Zoom:** Use zoom for detailed UI elements

### Script Format

Each script includes:
- **SCENE:** Scene number and description
- **VISUAL:** What appears on screen
- **NARRATION:** Voice-over text
- **ACTION:** Specific clicks, typing, or demonstrations
- **TIMING:** Approximate duration for each scene

### Accessibility

- Closed captions required for all videos
- Audio descriptions where appropriate
- High contrast visuals for clarity

## Production Workflow

1. **Script Review:** Technical review by engineering team
2. **Screen Recording:** Capture all demonstrations
3. **Narration Recording:** Professional studio recording
4. **Video Editing:** Combine visuals, narration, and effects
5. **Review Cycle:** Technical and brand review
6. **Captioning:** Add closed captions
7. **Final QA:** Complete quality check
8. **Publishing:** Deploy to learning platform

## File Naming Convention

```
greenlang_tutorial_[XX]_[topic]_v[version].[extension]

Examples:
- greenlang_tutorial_01_introduction_v1.0.mp4
- greenlang_tutorial_01_introduction_v1.0.srt
- greenlang_tutorial_01_introduction_v1.0_thumbnail.jpg
```

## Updating Videos

When GreenLang UI or features change:
1. Review affected scripts
2. Update script content
3. Re-record affected scenes
4. Re-edit video
5. Update captions
6. Republish with new version number
