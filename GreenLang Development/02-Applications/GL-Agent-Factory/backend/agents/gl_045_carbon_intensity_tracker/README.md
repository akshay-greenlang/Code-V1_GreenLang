# GL-045: Carbon Intensity Tracker Agent

Real-time carbon intensity monitoring per GHG Protocol and ISO 14064.

## Features

- Scope 1 (direct) emissions calculation
- Scope 2 (electricity) emissions calculation
- Carbon intensity per unit of production
- Decarbonization opportunity identification
- Support for custom emission factors

## Standards

- GHG Protocol: Greenhouse Gas Protocol
- ISO 14064: Greenhouse Gas Accounting and Verification
