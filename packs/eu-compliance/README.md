## Shared Dependencies

All EU Compliance Packs share:
- GreenLang Foundation Agents (orchestrator, schema, citations, audit trail)
- Authentication & RBAC (JWT, role-based access)
- Observability Stack (Prometheus, Grafana, OpenTelemetry)
- Data Quality Pipeline (validation, dedup, profiling)
- PostgreSQL + TimescaleDB + pgvector
