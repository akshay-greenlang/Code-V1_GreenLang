================================================================================
  API ENDPOINTS (14 TOTAL)
================================================================================

FACTOR QUERIES (5):
  GET  /api/v1/factors                  List factors (paginated)
  GET  /api/v1/factors/{id}             Get specific factor
  GET  /api/v1/factors/search           Full-text search
  GET  /api/v1/factors/category/{fuel}  Get by fuel type
  GET  /api/v1/factors/scope/{scope}    Get by scope

CALCULATIONS (5):
  POST /api/v1/calculate                Single calculation
  POST /api/v1/calculate/batch          Batch (max 100)
  POST /api/v1/calculate/scope1         Scope 1 emissions
  POST /api/v1/calculate/scope2         Scope 2 emissions
  POST /api/v1/calculate/scope3         Scope 3 (placeholder)

SYSTEM (4):
  GET  /api/v1/health                   Health check
  GET  /api/v1/stats                    Statistics
  GET  /api/v1/stats/coverage           Coverage stats
  GET  /                                Root endpoint

================================================================================
  QUICK START
================================================================================

1. Run with Docker Compose (recommended):
   cd greenlang/api
   docker-compose up -d

2. Access API:
   API: http://localhost:8000
   Docs: http://localhost:8000/api/docs

3. Test calculation:
   curl -X POST http://localhost:8000/api/v1/calculate \
     -H "Content-Type: application/json" \
     -d '{"fuel_type":"diesel","activity_amount":100,"activity_unit":"gallons"}'

4. Run tests:
   pytest greenlang/api/tests/ -v --cov
